# Getting Started with Windows Batch Scripting
---  
Windows批处理脚本是很容易访问到的。 - 它适用于任何现代Windows机器。 你可以在任何现代Windows机器上创建和修改批处理脚本。 这些工具从盒子里出来：Windows命令提示符和一个像Notepad.exe这样的文本编辑器。 这绝对不是最好的shell脚本语言，但它能完成了工作。
